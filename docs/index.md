# AI Usage Widget

AI Usage Widget is a small macOS menu-bar app and desktop widget for monitoring Claude and ChatGPT/Codex usage limits from the dashboards you already use.

The app is free and open source. It runs locally on your Mac, uses embedded WebKit pages for sign-in, and displays parsed usage summaries in the menu bar and WidgetKit widget.
